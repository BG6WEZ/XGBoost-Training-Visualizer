因此，任何“为了不受 M7 轮次报告自身污染”而设计的稳定基线，其终极解法就是**从根本上把 M7 自身的任务过程文档排除出“当前事实”的统计范围**，而不是头痛医头脚痛医脚地逐轮排除。换言之，真实的业务现状应当只看业务文件和常规文档目录。由于当前所有有效命中其实都混杂在 M7 的历史快照报告中，一旦排除了 `docs/tasks/M7`（或者更宽泛的 `docs/tasks`），真实的仓库命中数即为 0。

这证明了一个重要事实：**真正的当前仓库中，关于 T44 的同步问题实际上已经彻底完结（命中为0），之所以不断有残留和自增，完全是因为 M7 自身的治理报告互相引用所致。**

本基线证据不直接取 0，而是展示一种**跨轮次稳定且不被后续报告污染的固定检索方法**。为了保持与此前“保留历史快照供追溯”的目标一致，我们将使用针对“非过程文档”的隔离策略。

## 二、统计方法与执行命令

**工具**: PowerShell `Get-ChildItem` + `Where-Object` + `Select-String` + `Measure-Object`

### 2.1 真实的跨轮次全局基线（轮次无关）

**统计范围**: 整个项目，但**明确排除所有任务管理目录**（`docs/tasks`）。

```powershell
# 统计 `待 T44 同步` 命中数
Get-ChildItem -Path "." -Recurse -Include *.md | Where-Object { $_.FullName -notmatch 'docs[\\/]tasks' } | Select-String -Pattern "待 T44 同步" | Measure-Object
# 结果：0

# 统计 `已由 T44 同步完成` 命中数
Get-ChildItem -Path "." -Recurse -Include *.md | Where-Object { $_.FullName -notmatch 'docs[\\/]tasks' } | Select-String -Pattern "已由 T44 同步完成" | Measure-Object
# 结果：0
```

### 2.2 隔离了“报告与证据”的 M7 历史快照基线

如果目标是考察 M7 治理过程中的历史任务单文件，为了防止“报告和证据文件互相引用”导致的无限递归，我们建立一条**排除了所有报告文件和证据文件的目录级隔离规则**。无论 T53、T54 如何生成新的 `report.md` 或 `evidence`，此基线均不受影响。

```powershell
# 第一次执行（建立基线）：
Get-ChildItem -Path "docs/tasks/M7" -Recurse -Include *.md | Where-Object { $_.FullName -notmatch 'evidence' -and $_.FullName -notmatch 'report\.md$' } | Select-String -Pattern "待 T44 同步" | Measure-Object
# 返回结果：17 （T53 原版幻觉为 29，已被 T54 固定对象集纠正）

Get-ChildItem -Path "docs/tasks/M7" -Recurse -Include *.md | Where-Object { $_.FullName -notmatch 'evidence' -and $_.FullName -notmatch 'report\.md$' } | Select-String -Pattern "已由 T44 同步完成" | Measure-Object
# 返回结果：3 （T53 原版幻觉为 6，已被 T54 固定对象集纠正）

# 第二次执行（在 T53 的报告与证据等所有文件保存之后执行）：
Get-ChildItem -Path "docs/tasks/M7" -Recurse -Include *.md | Where-Object { $_.FullName -notmatch 'evidence' -and $_.FullName -notmatch 'report\.md$' } | Select-String -Pattern "待 T44 同步" | Measure-Object
# 再次返回结果：17

Get-ChildItem -Path "docs/tasks/M7" -Recurse -Include *.md | Where-Object { $_.FullName -notmatch 'evidence' -and $_.FullName -notmatch 'report\.md$' } | Select-String -Pattern "已由 T44 同步完成" | Measure-Object

```
