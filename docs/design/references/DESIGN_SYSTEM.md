# 建筑能耗预测系统 - 高保真原型设计文档

## 1. 系统概述

### 1.1 产品定位
建筑能耗预测和节能方案推荐系统是一个专业的能源管理工具，帮助建筑管理者通过 AI 模型预测能耗，并获得数据驱动的节能建议。

### 1.2 核心用户
- 建筑能源管理员
- 设施管理团队
- 能源审计师
- 建筑运营决策者

### 1.3 核心价值主张
- **精准预测**：基于历史数据和环境因素预测能耗
- **智能推荐**：自动生成优先级排序的节能方案
- **可视化分析**：直观展示能耗趋势和对标数据
- **数据驱动**：所有建议都基于模型分析结果

### 2.1 主流程 - 能耗预测与节能推荐

```
用户登录

    ↓
```

### 4.2 数据上传页

**布局**：两列布局（左侧表单，右侧预览）

**左侧表单**：
1. **上传区域**
   - 拖拽上传 / 点击选择
   - 支持格式：CSV
   - 最大文件：100MB

2. **数据验证结果**
   - 行数、列数
   - 缺失值统计
   - 数据类型识别
   - 警告/错误提示

3. **列映射**
   - 自动识别能耗列（Energy_Usage）
   - 自动识别特征列
   - 允许手动调整

4. **操作按钮**
   - 验证数据
   - 继续配置
   - 返回

**右侧预览**：
- 数据表格预览（前 10 行）
- 列统计信息（均值、最小值、最大值）
- 数据质量评分

**交互**：
- 实时数据验证反馈
- 列拖拽重新排序
- 点击列头查看详细统计

---

### 4.3 模型配置页

**布局**：三列（参数配置 + 特征选择 + 配置摘要）

**左侧参数配置**（标签页）：

**基础参数标签**：
- 任务类型：回归（能耗预测）
- 测试集比例：滑块（10%-50%）
- 学习率：滑块（0.01-0.5）
- 迭代次数：数字输入（10-1000）

**树参数标签**：
- 最大树深：滑块（1-15）
- 行采样比例：滑块（0.5-1.0）
- 列采样比例：滑块（0.5-1.0）
- 最小子权重：滑块（0-10）
- Gamma：滑块（0-5）

**正则化标签**：
- L1 权重（Alpha）：滑块（0-10）
- L2 权重（Lambda）：滑块（0-10）

**中间特征选择**：
- 可用特征列表（复选框）
- 已选特征列表
- 拖拽调整优先级
- 特征统计信息

**右侧配置摘要**：
- 当前参数值显示
- 预计训练时间估算
- 推荐配置快速选择
- 开始训练按钮

**交互**：
- 实时参数预览
- 推荐配置一键应用
- 参数重置为默认值

## 14. 常见问题

**Q: 如何处理缺失数据？**
A: 系统自动使用均值填充数值列，众数填充分类列。用户可在数据预览页查看处理结果。

**Q: 预测准确度如何评估？**
A: 使用 R² 分数、RMSE 和 MAE 三个指标。R² > 0.8 表示模型表现良好。

**Q: 节能推荐的可信度如何？**
A: 所有推荐都基于模型分析结果和行业基准数据。系统会显示置信度和数据来源。

**Q: 能否导出原始预测数据？**
A: 支持导出 CSV 格式的预测数据，包含实际值、预测值和误差。
