# 建筑能耗预测系统 - 完整设计系统规范

## 1. 设计哲学

本设计系统遵循以下核心原则：**优雅、专业、数据驱动、易用性**。采用深色主题设计，既能减少眼睛疲劳，又能突出数据可视化的重点。

---

## 2. 色彩系统

### 2.1 核心色彩板

**背景色系**
- 主背景：`#0F1419`（深蓝黑）- 用于页面背景
- 次背景：`#1A1F2E`（深灰蓝）- 用于卡片和容器
- 三级背景：`#242B3A`（更浅的灰蓝）- 用于悬停和激活状态

**文字色系**
- 主文字：`#E5E7EB`（浅灰）- 用于主要文本
- 次文字：`#9CA3AF`（中灰）- 用于辅助文本
- 禁用文字：`#6B7280`（深灰）- 用于禁用状态

**功能色系**
- 主色（蓝紫）：`#6366F1` - 用于主要操作、链接、强调
- 成功（绿色）：`#10B981` - 用于成功状态、完成指示
- 警告（橙色）：`#F59E0B` - 用于警告和注意
- 错误（红色）：`#EF4444` - 用于错误和危险操作
- 信息（蓝色）：`#3B82F6` - 用于信息提示

### 2.2 色彩使用规则

| 场景 | 颜色 | 用途 |
|------|------|------|
| 主要按钮 | `#6366F1` | 重要操作（上传、训练、提交） |
| 次要按钮 | `#3B82F6` | 次要操作（取消、返回） |
| 成功反馈 | `#10B981` | 训练完成、数据上传成功 |
| 进行中 | `#3B82F6` | 训练中、加载中 |
| 错误提示 | `#EF4444` | 验证失败、训练失败 |
| 警告信息 | `#F59E0B` | 数据质量问题、异常值 |
| 链接文字 | `#6366F1` | 可点击的文本链接 |

### 2.3 渐变色系

用于卡片背景和视觉强调：
- 主渐变：从 `#6366F1` 到 `#3B82F6`（蓝紫到蓝）
- 成功渐变：从 `#10B981` 到 `#059669`（浅绿到深绿）
- 警告渐变：从 `#F59E0B` 到 `#D97706`（浅橙到深橙）

## 3. 排版系统

### 3.1 字体选择

**主字体**：Inter（用于所有 UI 文本）
- 特点：清晰、现代、易读
- 权重：400（Regular）、500（Medium）、600（Semibold）、700（Bold）

**代码字体**：Fira Code（用于代码和技术文本）
- 特点：等宽、清晰、专业

### 3.2 排版规模

| 层级 | 字体大小 | 行高 | 字重 | 用途 | 示例 |
|------|---------|------|------|------|------|
| H1 | 32px | 40px | 700 | 页面标题 | "建筑能耗预测平台" |
| H2 | 24px | 32px | 700 | 区域标题 | "训练结果分析" |
| H3 | 18px | 28px | 600 | 卡片标题 | "模型性能指标" |
| H4 | 16px | 24px | 600 | 小标题 | "特征重要性" |
| Body Large | 16px | 24px | 400 | 主要文本 | 描述文字 |
| Body | 14px | 22px | 400 | 正文内容 | 卡片内容 |
| Body Small | 12px | 18px | 400 | 辅助文本 | 时间戳、标签 |
| Caption | 11px | 16px | 400 | 最小文本 | 图表标签 |

### 3.3 字距和段落

- 字间距（Letter spacing）：正常（0）
- 段落间距：16px（Body 文本之间）
- 列表项间距：8px

## 4. 间距系统

基础间距单位：**4px**

| 名称 | 值 | 用途 |
|------|-----|------|
| XS | 4px | 紧凑间距（按钮内部、图标间距） |
| SM | 8px | 小间距（表单字段、列表项） |
| MD | 12px | 中间距（卡片内部间距） |
| LG | 16px | 大间距（区域间距、卡片间距） |
| XL | 24px | 超大间距（页面区域间距） |
| 2XL | 32px | 最大间距（主要区域分隔） |
| 3XL | 48px | 特大间距（页面顶部间距） |

### 4.1 常见间距组合

- 卡片内部 Padding：16px（所有方向）
- 卡片间距 Gap：16px
- 表单字段间距：12px
- 按钮内部 Padding：12px（水平）× 8px（垂直）
- 页面顶部 Margin：48px
- 页面左右 Padding：24px

## 8. 动画和过渡

### 8.1 过渡时间

| 类型 | 时间 | 用途 |
|------|------|------|
| 快速 | 150ms | 悬停、焦点状态 |
| 标准 | 300ms | 页面过渡、模态框 |
| 缓慢 | 500ms | 复杂动画、加载状态 |

### 8.2 缓动函数

- 默认：`cubic-bezier(0.4, 0, 0.2, 1)`（ease-in-out）
- 进入：`cubic-bezier(0, 0, 0.2, 1)`（ease-out）
- 退出：`cubic-bezier(0.4, 0, 1, 1)`（ease-in）

### 8.3 常见动画

**淡入淡出**
```css
transition: opacity 300ms ease-in-out;
```

**滑动**
```css
transition: transform 300ms ease-in-out;
```

**缩放**
```css
transition: transform 300ms ease-in-out, box-shadow 300ms ease-in-out;
```

**颜色变化**
```css
transition: background-color 300ms ease-in-out, color 300ms ease-in-out;
```

## 9. 组件规范

### 9.1 按钮

**按钮大小**
- Small：32px 高，12px 水平 Padding
- Medium：40px 高，16px 水平 Padding
- Large：48px 高，20px 水平 Padding

**按钮状态**
- 默认：`#6366F1` 背景，`#E5E7EB` 文字
- 悬停：背景变深 10%，阴影升级到 MD
- 激活：背景变深 20%
- 禁用：`#6B7280` 背景，`#9CA3AF` 文字，无阴影

**按钮变体**
- Primary：实心蓝紫色
- Secondary：实心蓝色
- Outline：透明背景，彩色边框
- Ghost：无背景无边框，仅文字
- Danger：实心红色

### 9.2 输入框

**输入框高度**：40px  
**输入框 Padding**：12px（水平）× 8px（垂直）  
**输入框边框**：1px，`#374151` 颜色  
**输入框圆角**：4px

**输入框状态**
- 默认：灰色边框
- 焦点：蓝紫色边框（2px），阴影 SM
- 错误：红色边框，错误提示文字
- 禁用：灰色背景，禁用文字

### 9.3 卡片

**卡片 Padding**：16px  
**卡片边框**：1px，`#374151` 颜色  
**卡片圆角**：8px  
**卡片阴影**：SM 阴影  
**卡片背景**：`#1A1F2E`

**卡片变体**
- 默认：标准卡片
- Elevated：MD 阴影，提升视觉层级
- Outlined：无阴影，强调边框
- Filled：背景色更深

### 9.4 标签/徽章

**标签大小**：24px 高，8px 水平 Padding  
**标签圆角**：4px  
**标签字体**：12px，600 字重

**标签颜色**
- 默认：灰色背景
- Primary：蓝紫色背景
- Success：绿色背景
- Warning：橙色背景
- Error：红色背景

## 11. 无障碍设计

### 11.1 对比度要求

- 文字与背景：最小 4.5:1（WCAG AA）
- 图形元素：最小 3:1（WCAG AA）

### 11.2 焦点指示器

- 焦点颜色：`#6366F1`
- 焦点宽度：2px
- 焦点样式：外部边框

### 11.3 键盘导航

- Tab 键：顺序导航
- Shift + Tab：反向导航
- Enter：激活按钮
- Space：复选框、单选框
- Escape：关闭模态框

## 15. 微交互

### 15.1 悬停效果

- 按钮：背景变深，阴影升级
- 卡片：阴影升级，轻微缩放（1.02x）
- 链接：下划线出现，颜色变浅

### 15.2 加载状态

- 加载动画：旋转 360°，2 秒一周期
- 骨架屏：浅色脉冲动画，1.5 秒周期
- 进度条：平滑过渡，无跳跃

### 15.3 反馈动画

- 成功：绿色闪烁，300ms
- 错误：红色抖动，200ms × 3 次
- 警告：橙色脉冲，500ms
