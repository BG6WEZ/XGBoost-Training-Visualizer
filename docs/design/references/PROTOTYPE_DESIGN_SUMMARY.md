## 设计特点

### 1. 用户中心设计

**用户研究**
- 识别 3 个主要用户角色
- 定义每个角色的目标和需求
- 创建完整的用户旅程地图

**可用性考虑**
- 简化复杂的机器学习概念
- 提供清晰的参数说明和预设配置
- 实时反馈和进度展示
- 详细的错误提示和恢复方案

### 2. 视觉设计

**深色主题**
- 减少眼睛疲劳，适合长时间使用
- 突出数据可视化的重点
- 现代、专业的视觉风格

**色彩系统**
- 15+ 精心选择的颜色
- 功能色彩清晰（成功、警告、错误）
- 符合 WCAG AA 无障碍标准

**排版系统**
- 8 个排版规模，清晰的层级
- 两种字体（UI 和代码）
- 充分的行高和字间距

### 3. 交互设计

**微交互**
- 悬停效果（阴影升级、背景变深）
- 加载动画（旋转、脉冲）
- 反馈动画（闪烁、抖动）

**过渡动画**
- 页面进入：淡入 + 向上滑动（300ms）
- 页面退出：淡出 + 向下滑动（300ms）
- 模态框：缩放 + 淡入（300ms）

**响应式设计**
- 3 个断点（移动、平板、桌面）
- 流体布局，自适应内容
- 触摸友好的交互区域

### 4. 无障碍设计

**键盘导航**
- Tab 键顺序导航
- 焦点指示器清晰可见
- 快捷键支持

**屏幕阅读器**
- 所有按钮都有 aria-label
- 表单字段都有关联的 label
- 图像都有 alt 文本

**视觉对比**
- 所有文字对比度 ≥ 4.5:1（WCAG AA）
- 不仅依赖颜色传达信息
- 清晰的焦点指示器

## 核心页面设计亮点

### 1. 仪表板页面

**设计亮点**
- 4 张统计卡片，快速了解关键指标
- 任务列表卡片式布局，状态一目了然
- 状态指示器（绿色完成、蓝色进行、红色失败）
- 实时数据更新，骨架屏加载动画

### 2. 数据上传页面

**设计亮点**
- 拖拽上传区域，大而清晰
- 多步骤表单，逐步引导用户
- 列映射表格，直观配置
- 数据质量报告，显示数据问题和改进建议

### 3. 模型配置页面

**设计亮点**
- 左右两列布局，参数配置 + 说明
- 滑块 + 数字输入，双向交互
- 参数说明动态显示，根据焦点字段更新
- 预设配置，一键应用常见配置

### 4. 训练监控页面

**设计亮点**
- 大进度条，清晰显示训练进度
- 实时图表，动态更新损失曲线
- 实时指标卡片，显示关键数据
- 训练日志，记录详细的训练过程

### 5. 结果分析页面

**设计亮点**
- 性能指标卡片，突出关键指标
- 特征重要性图表，显示特征贡献度
- 混淆矩阵/残差分析，诊断模型性能
- 历史模型对比，追踪模型改进

### 6. 节能推荐页面

**设计亮点**
- 推荐卡片按优先级排序
- 优先级边框颜色区分（红/橙/蓝）
- 4 个关键指标（节能潜力、难度、ROI、回本周期）
- 3-5 个具体行动项，可执行性强

## 技术栈建议

### 前端

- **框架**：React 19 + TypeScript
- **样式**：Tailwind CSS 4
- **组件库**：shadcn/ui
- **图表**：Recharts
- **图标**：Lucide React
- **动画**：Framer Motion
- **表单**：React Hook Form + Zod
- **状态管理**：TanStack Query
- **路由**：React Router v6

### 后端

- **框架**：Express.js + Node.js
- **API**：tRPC
- **数据库**：MySQL/PostgreSQL
- **ORM**：Drizzle ORM
- **认证**：JWT + OAuth
- **文件存储**：S3

### 机器学习

- **模型**：XGBoost
- **数据处理**：Pandas + NumPy
- **可视化**：Matplotlib + Plotly
- **部署**：Docker + Kubernetes
