# 建筑能耗预测与节能方案推荐系统 - 高保真原型设计

## 项目概述

本项目是一个**专业级的建筑能耗预测和节能方案推荐平台**，采用优雅的深色主题设计，为建筑能源管理人员提供数据驱动的节能决策支持。

### 核心价值主张

- **精准预测** - 基于 XGBoost 机器学习模型，预测建筑能耗趋势
- **智能推荐** - 根据数据分析自动生成个性化节能方案
- **可视化分析** - 直观展示能耗数据、模型性能和节能潜力
- **对标管理** - 与行业基准对比，识别能耗异常

---

## 设计理念

### 视觉风格
- **主题** - 深色优雅主题，适合长时间数据分析工作
- **配色** - 深蓝/紫色主色调，绿色/红色用于状态指示
- **排版** - 清晰的信息层级，使用 Inter/Poppins 字体
- **组件** - 现代化卡片设计，微妙阴影和渐变效果

### 交互原则
- **流畅性** - 平滑的页面过渡和动画反馈
- **清晰性** - 明确的操作路径和视觉反馈
- **效率性** - 最少化用户操作步骤，快速访问关键功能
- **可访问性** - 支持键盘导航，高对比度文本

## 用户角色

### 1. 能源管理员（Primary User）
- 上传建筑能耗数据
- 配置模型训练参数
- 监控训练进度
- 查看预测结果和节能推荐
- 导出报告和数据

### 2. 建筑运维人员
- 查看能耗预测和趋势
- 了解节能方案和实施建议
- 跟踪节能效果

### 3. 管理层决策者
- 查看能耗对标和行业排名
- 了解节能投资回报率
- 获取高层次的能耗概览

## 核心功能模块

### 模块 1：数据管理
**页面** - 数据上传与预览  
**功能**
- 支持 CSV 文件上传（单文件或文件夹批量）
- 自动列检测和映射
- 数据质量检查（缺失值、异常值）
- 数据预览和统计

### 模块 2：模型训练
**页面** - 参数配置与训练监控  
**功能**
- 灵活的 XGBoost 超参数配置
- 实时训练进度显示
- 损失曲线和指标动态更新
- 训练历史记录

### 模块 3：结果分析
**页面** - 预测结果与性能评估  
**功能**
- 模型性能指标展示
- 特征重要性排序
- 混淆矩阵/残差分析
- 预测结果可视化

### 模块 4：节能推荐
**页面** - 智能推荐与方案详情  
**功能**
- 5 大节能推荐规则
- 优先级排序和 ROI 计算
- 具体行动项和实施指南
- 节能潜力评估

### 模块 5：对标分析
**页面** - 能耗对标与对比  
**功能**
- 多建筑能耗对比
- 行业基准对标
- 能耗趋势分析
- 排名和排序

### 模块 6：数据管理
**页面** - 历史记录与导出  
**功能**
- 训练任务历史
- 结果对比和版本管理
- 报告导出（PDF/Excel）
- 数据下载

## 交互流程

### 用户流程 1：首次使用 - 完整训练流程
```
登录 → 仪表板 → 上传数据 → 配置参数 → 开始训练 → 监控进度 → 查看结果 → 查看推荐 → 导出报告
```

### 用户流程 2：查看历史结果
```
仪表板 → 历史记录 → 选择任务 → 查看结果 → 对比分析
```

### 用户流程 3：对标分析
```
仪表板 → 对标分析 → 选择建筑 → 查看对比 → 导出对标报告
```

## 设计规范

### 色彩系统
| 用途 | 颜色 | 十六进制 |
|------|------|---------|
| 背景 | 深蓝黑 | #0F1419 |
| 卡片 | 深灰蓝 | #1A1F2E |
| 主色 | 蓝紫 | #6366F1 |
| 成功 | 绿色 | #10B981 |
| 警告 | 橙色 | #F59E0B |
| 错误 | 红色 | #EF4444 |
| 文字 | 浅灰 | #E5E7EB |

### 排版系统
| 层级 | 字体大小 | 字重 | 用途 |
|------|---------|------|------|
| H1 | 32px | 700 | 页面标题 |
| H2 | 24px | 700 | 区域标题 |
| H3 | 18px | 600 | 卡片标题 |
| Body | 14px | 400 | 正文内容 |
| Small | 12px | 400 | 辅助文本 |

### 间距系统
- 基础单位：4px
- 常用间距：8px, 12px, 16px, 24px, 32px, 48px

### 圆角系统
- 小组件：4px
- 卡片：8px
- 模态框：12px

### 阴影系统
- 浅阴影：0 1px 3px rgba(0,0,0,0.12)
- 中阴影：0 4px 12px rgba(0,0,0,0.15)
- 深阴影：0 8px 24px rgba(0,0,0,0.20)

## 响应式设计

### 断点
- 移动端：< 640px
- 平板：640px - 1024px
- 桌面：> 1024px

### 设计原则
- 移动优先
- 触摸友好（最小 44px 点击区域）
- 流体布局
- 隐藏非关键信息
