# XGBoost 训练可视化工具 - UI 设计规范

## 1. 设计系统

### 1.1 设计理念

本设计系统遵循**专业、高效、数据驱动**的设计原则，采用深色主题减少眼睛疲劳，突出数据可视化效果，适合数据科学团队的长时间使用。

| 原则 | 描述 |
|------|------|
| **专业性** | 清晰的信息层级，精准的数据展示 |
| **高效性** | 最小化操作步骤，快捷访问核心功能 |
| **一致性** | 统一的组件规范，可预测的交互模式 |
| **可访问性** | WCAG AA 级别对比度，键盘导航支持 |
